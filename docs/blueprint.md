# **App Name**: EventZenith

## Core Features:

- User Registration Form: Collect user information using a multi-page form.
- Input Validation: Validate each input field in real-time using JavaScript to ensure data quality and proper formatting.
- Progress Indicator: Show progress through the multiple steps.
- Database Storage: Store validated user data into a SQLite database using a Python backend for persistence.
- Confirmation Page: Display a summary of the entered data for review before final submission.

## Style Guidelines:

- Primary color: Deep blue (#34495E) to convey trust and reliability.
- Background color: Light gray (#F2F3F4) to provide a clean, neutral backdrop.
- Accent color: Teal (#2ECC71) to highlight important actions and provide visual interest.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and a modern touch.
- Use clean, simple icons from a set like FontAwesome for form elements.
- Employ Bootstrap's grid system to create a responsive and user-friendly layout across different devices.
- Use subtle transition effects on form elements to give the interface a smooth feel.
